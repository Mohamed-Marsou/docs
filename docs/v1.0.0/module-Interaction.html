Interaction module providing utility functions for interacting with web pages. This module includes functions for selecting dropdown options, handling scrolling behavior, and interacting with elements via CSS selectors or XPath. It is designed to support various interaction scenarios, including random scrolling and precise element targeting.

(inner) scrollToElement(element, maxScrollingTimeopt, minScrollingTimeopt, maxScrollingSpeedopt, minScrollingSpeedopt, offsetopt) → {Promise.<void>}

Scrolls randomly to a specific element on the page. This function smoothly scrolls to an element using either a CSS selector or an XPath. It simulates human-like scrolling behavior by introducing random scrolling movements before reaching the final position.
Parameters:
NameTypeAttributesDefaultDescription
elementObject | nullThe target element to scroll to.
Properties
NameTypeDescription
selectorstringThe selector of the target element (CSS or XPath).
typestringThe type of selector ('css' or 'xpath').
maxScrollingTimenumber<optional>
5000Maximum scrolling duration in milliseconds.
minScrollingTimenumber<optional>
3000Minimum scrolling duration in milliseconds.
maxScrollingSpeednumber<optional>
900Maximum scrolling speed delay in milliseconds.
minScrollingSpeednumber<optional>
500Minimum scrolling speed delay in milliseconds.
offsetnumber<optional>
500Adjusts scroll position relative to the target element.
Throws:
- Throws an error if the element cannot be located.
Type
Error
Returns:
- Resolves when scrolling is completed.
Type: 
Promise.<void>
Example
// Scroll to an element using XPath
const targetElement = {
  selector: '//div[@id="target"]',
  type: 'xpath'
};
await WP.scrollToElement(targetElement);

// Scroll to an element using a CSS selector
const targetElement = {
  selector: '#target-element',
  type: 'css'
};
await WP.scrollToElement(targetElement, 7000, 4000, 1000, 600, 300);

Properties

Type
Error
Returns:
- Resolves when the option is successfully selected.
Type: 
Promise.<void>
Examples
// Select an option by value using a CSS selector
await utilities.selectOption({
    selector: '#dropdown-id',
    type: 'selector'
}, 'option-value');
// Select an option by value using an XPath

    selector: '//select[@id="dropdown-id"]',
    type: 'xpath'
}, 'option-value');
