selectOption( params, optionValue )

Description
Selects an option from a dropdown menu by its value. This function uses Puppeteer's `page.select()` method to select an option in a dropdown element. It accepts either a CSS selector or an XPath to locate the dropdown and selects the desired option based on the provided value.
Parameters
Name Type Description
params Object The parameters object.
Name Type Attributes Default Description
selector string The CSS selector or XPath of the dropdown element.
type string <optional>
 'selector' The type of selector used ('selector' or 'xpath').
optionValue string The value of the option to be selected.
Examples
// Select an option by value using a CSS selector
			await utilities.selectOption({
			    selector: '#dropdown-id',
			    type: 'selector'
			}, 'option-value');
			
			// Select an option by value using an XPath
			await utilities.selectOption({
			    selector: '//select[@id="dropdown-id"]',
			    type: 'xpath'
			}, 'option-value');
Throws
If the dropdown is not found or the selection fails.
